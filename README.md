# Hidden Line

Hidden Line is a mobile-first precision tracing game. The player follows a hidden seed-generated path by revealing only the area around their fingertip.

## Live Demo

- GitHub Pages: <https://kangsungbae87.github.io/hiddenline/>

## Local Development

```bash
npm install
npm run dev
```

## Verification

```bash
npm test
npm run build
npm run e2e
```

## Deployment

The app is built with `GITHUB_PAGES=true npm run build` and published from the `dist` output to the `gh-pages` branch.
